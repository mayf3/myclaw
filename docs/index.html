总诊断

MyClaw 不应该复刻 Hermes、OpenClaw 或 OpenHuman。推荐先做 Node.js/TypeScript 的 本地工作流内核，吸收 OpenClaw Lobster 的 typed workflow、JSON envelope、 approval/resume，再吸收 OpenHuman 的 controller registry、event bus、 tool permission。Phase 1.2 已在 Human Experiments、共享 route adapter 和 approval queue 之上， 补齐强制结构红线、生成物 stale 检查、HTML Center 仓库命令和 L0-L6 分层测试路线； Feishu/Lark 仍参考 OpenClaw `extensions/feishu`，但不直接加载插件 runtime。
